In this release we have added GUI support for browsing and interacting with the
HDFS, running in local workstation. Following are the highlights of this
feature.

- Support for uploading multiple files to any directory.
- Create, rename and delete directories and files in HDFS.
- Multi-column directory layout for fast and easy navigation
